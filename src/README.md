# Whippet
## Static site generator

## This module is still under construction and is extremely unstable!

## Usage

Install
```npm install -g```

Create a new site
```whippet new mySite```

Change to the site directory
```cd mySite```

Build the site
```whippet build```

or

Watch the folder for changes and automatically build the site
```whippet watch```
